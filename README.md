Microservicio que expone funcionalidades principalmente relacionadas con la consulta de información consolidada para reportes administrativos, usuarios, cursos, estados de cursos y evaluaciones.
Principales métodos:
- Lista todos los usuarios registrados.
- Busca y retorna un usuario por su ID.
- Retorna un reporte consolidado de un usuario, incluyendo: Sus cursos y el estado de cada uno; Sus evaluaciones y las notas obtenidas.
- Lista todos los cursos.
- Busca un curso por su ID.
- Lista los cursos filtrados por un estado específicoç
- Lista todos los estados de cursos.
- Busca un estado de curso por su ID.
- Cuenta cuántos usuarios han finalizado un curso específico.
- Cuenta cuántos usuarios tienen pendiente un curso específico.
- Devuelve una lista de todas las evaluaciones.
- Devuelve la evaluación con el ID indicado.
- Lista todas las notas de evaluaciones.
- Devuelve la nota de evaluación con el ID indicado.
- Devuelve la cantidad de usuarios que rindieron una evaluación específica.
